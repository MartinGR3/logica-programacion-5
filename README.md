# Lógica de Programación 5

## Descripción
Este proyecto consiste en un mini diccionario Español-Inglés desarrollado en Java.

El programa:
- Guarda palabras y sus traducciones utilizando un `HashMap`.
- Selecciona aleatoriamente 5 palabras en español.
- Solicita al usuario escribir la traducción en inglés.
- Verifica si las respuestas son correctas.
- Muestra al final la cantidad de respuestas correctas e incorrectas.

---

## Tecnologías utilizadas
- Java
- IntelliJ IDEA
- HashMap
- Programación orientada a objetos

## Funcionalidades
1. Diccionario Español-Inglés con más de 20 palabras.
2. Selección aleatoria de preguntas. 
3. Validación de respuestas. 
4. Conteo de aciertos y errores.

## Cómo ejecutar el proyecto
- Clonar el repositorio:
git clone https://github.com/MartinGR3/logica-programacion-5.git
- Abrir el proyecto en IntelliJ IDEA. 
- Ejecutar la clase:
`LogicaProgramacion5Main.java`

